Search Image

  • To search for similar images, upload an image along with how many results (K) and minimum similarity (Threshold) you want using the search function on our website.

Train Model

  • Click the "Train" button and select the folder containing your images. The system will learn from these images to improve its ability to recognize similar ones in the future.

Selective Folder Path

  • To update the image database, upload your images to a new folder and send a request to the website using the 'rebuild' function. Provide the folder path where your images are located.

Insert Image

  • To add an image to the system, upload it using the website's uploader.

Insert Folder

  • To add a new folder of images to the index, use the upload tool and select the folder you want to add.

Delete Images

  • To remove images from the system, use the "Delete" function and provide a list of image file names (e.g., "image1.jpg", "image2.png") in the body of your request.
